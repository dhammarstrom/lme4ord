Simple toy examples -- diagonal matrices

The simplest structured matrix is probably the identity, which one may produce with the following code.

(ident <- strucMatrixIdent(matSize = 5))

## Structured sparse matrix
## ------------------------
## dimensions:
##  5 rows and 5 columns
##  5 nonzero values
##  1 repeated values
## 
## repeated values:
## [1] 1
## 
## row, column, and value indices (with associated values):
##   rowInds colInds valInds vals
## 1       0       0       1    1
## 2       1       1       1    1
## 3       2       2       1    1
## 4       3       3       1    1
## 5       4       4       1    1
## 
## special structured sparse matrix, inheriting from:
## [1] "strucMatrixIdent" "strucMatrixDiag"  "strucMatrix"

image(ident, plain = TRUE)

The ident object is indeed the five dimensional identity matrix, as can be seen by coercing it to a standard R matrix object.

as.matrix(ident)

##      [,1] [,2] [,3] [,4] [,5]
## [1,]    1    0    0    0    0
## [2,]    0    1    0    0    0
## [3,]    0    0    1    0    0
## [4,]    0    0    0    1    0
## [5,]    0    0    0    0    1

image(ident)

The difference between ident and as.matrix(ident) is in how they are stored. In particular, whereas as.matrix(ident) is stored as a general matrix, ident is stored in a way that takes its structure into account. In particular, ident is an R list,

is.list(ident)

## [1] TRUE

with the following five elements,

names(ident)

## [1] "rowInds" "colInds" "valInds" "vals"    "trans"

Row and column indices

The first two elements give zero-based row and column indices for each non-zero element in the matrix.

ident[c("rowInds", "colInds")]

## $rowInds
## [1] 0 1 2 3 4
## 
## $colInds
## [1] 0 1 2 3 4

Unique values

The vals element gives the unique non-zero values, which in this case is just the single number 1.

ident$vals

## [1] 1

Unique value indices

The valInds element gives a vector of length equal to rowInds and colInds for identifying the elements of val that correspond to each non-zero element. In this case, valInds is particularly simple and is just a vector of 1s.

ident$valInds

## [1] 1 1 1 1 1

To better understand what these values mean, consider a general diagonal matrix.

diagonal <- strucMatrixDiag(vals = (5:1)/10)
