lme4ord

Generalized linear mixed models with flexible covariance structures
lme4ord runs on top of the well-tested c++ machinery from lme4

Learn more »

For analysts

lme4ord provides a formula interface for specifying generalized linear mixed models with flexible covariance structures, in a manner similar to varStruct and corStruct classes in nlme.

Example »

For developers

lme4ord provides infrastructure for specifying new types of covariance structures, including a sparse matrix class that simplifies the specification of covariance structures and a flexible formula parsing paradigm that alleviates the need to fight with R expressions.

Example »

For statisticians

lme4ord also provides more direct access to the underlying lme4 machinery, for those comfortable with specifying their model as a series of sparse matrices and updating functions. This option is analogous to using glm.fit instead of glm.
